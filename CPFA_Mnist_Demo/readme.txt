=============================================================================================================
Python code for the paper "Convolutional Poisson Gamma Belief Network" published in ICML2019

=============================================================================================================
CPFA_Mnist_Demo folder contains 4 different training algorithms for CPFA:

CPFA_Mnist_Batch_Matrix_Slow(Toeplitz)

CPFA_Mnist_Batch_Elements_Fast(Element)

CPFA_Mnist_Batch_Elements_Par(Element-Parallel)

CPFA_Mnist_Par_Online(SGMCMC)

=============================================================================================================
By comparing the above demos, you can understand the necessity of optimizing the model training algorithm.
